# The Codex is a repository that gathers examples of some Arrays manipulations, Bits manipulations, Mathematical algorithms and more, written in JavaScript, Python, C++, Haskell.


# TRANSLATR, a command line interface to translate between different languages in the terminal.


# JS-Password-Generator, a simple password generator with JS, HTML and CSS.


# dec_input_converter, a decimal to binary, hexadecimal, octal, ASCII converter.


# SnakeJS, a simple snake game using only javascript, html and css, with the score incrementing by 2^n-1.


# JSTextEditor, a simple text editor made with JavaScript, HTML and CSS.


# uther-minal, a small JavaScript animation in the terminal.


# brute_pdf, a simple python brute-force for any password-protected pdf file, with pikepdf lib.


# lazy-clone, a simple bash script that automates the git clone.
